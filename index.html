Introdução do que é HTML

HTML ou HyperText Markup Language é a linguagem padrão utilizada para criar e estruturar páginas na web. É uma linguagem de marcação que define a estrutura básica do conteúdo de um site através de elementos como títulos, parágrafos, imagens, links e outros componentes.

Com HTML, é possível organizar e formatar informações de maneira que possam ser exibidas de forma adequada pelos navegadores.

Historia da WEB

A história da web começou em 1989, quando Tim Berners-Lee, um cientista da computação britânico, propôs um sistema de gerenciamento de informações que viria a ser conhecido como a World Wide Web (WWW). Em 1991, ele lançou a primeira página web e, em 1993, o Mosaic, o primeiro navegador gráfico, foi desenvolvido, facilitando o acesso e a navegação pela internet. Desde então, a web evoluiu rapidamente, transformando-se em uma plataforma global essencial para comunicação, comércio, educação e entretenimento.

O que são Clients

Em computação, um cliente é um dispositivo ou programa que solicita serviços ou recursos de um servidor. Na arquitetura cliente-servidor, o cliente é geralmente o navegador web (como Chrome, Firefox, ou Safari) que faz requisições a um servidor para acessar páginas web, baixar arquivos, enviar dados de formulário, entre outras interações. O cliente é responsável por apresentar a interface ao usuário final e enviar as solicitações ao servidor.

O que são Servers

Um servidor é um computador ou programa que fornece serviços, recursos ou dados a clientes através de uma rede. Na web, um servidor web é responsável por hospedar sites e responder às requisições feitas pelos navegadores (clientes). Ele processa as solicitações recebidas, recupera ou gera o conteúdo solicitado e o envia de volta ao cliente para ser exibido. Exemplos de servidores web incluem Apache, Nginx e Microsoft IIS.

Linguagens de programação

Frontend

  1. HTML: Utilizado para a estruturação do conteúdo na web.
  2. CSS: Usado para estilizar e layout das páginas web.
  3. JavaScript: Linguagem de programação que permite adicionar interatividade às páginas web.
  4. TypeScript: Um superset do JavaScript que adiciona tipagem estática opcional.
  5. React: Uma biblioteca JavaScript para construir interfaces de usuário.
  6. Angular: Um framework JavaScript para desenvolvimento de aplicações web dinâmicas.
  7. Vue.js: Um framework JavaScript progressivo para construir interfaces de usuário.

Backend

  1. Node.js: Permite executar JavaScript no lado do servidor.
  2. Python: Usado com frameworks como Django e Flask para desenvolvimento web.
  3. Ruby: Usado com o framework Ruby on Rails.
  4. PHP: Uma linguagem de script amplamente utilizada para desenvolvimento web.
  5. Java: Usado com frameworks como Spring para desenvolvimento de aplicações web robustas.
  6. C#: Utilizado com o framework ASP.NET da Microsoft.
  7. Go: Conhecida por sua eficiência e desempenho em servidores web.
